# Cardio-Man
#### CPSC 386 Project 1 - Programmed by Kinsey Vo and Anthony Tran

Game: A character running across the screen collecting items that will help or harm them.

## Items
  - Coin: +5 points
  - Health Box: + 20 HP
  - Spike: - 20 HP
